## 📢 BROADCAST SYSTEM
- Admin/Owner can use /broadcast
- Bot asks for:
  - text message
  - or photo + caption
- Sends to all registered users
- Shows:
  - sent count
  - failed count
---

## 🚫 BAN / UNBAN SYSTEM
- /ban <user_id> → bans user
- /unban <user_id> → unbans user
- Banned users cannot interact with the bot
- Banned users see a ban message when trying to use the bot

---

## 📊 ADMIN REPORTS
- /taskstats → show how many tasks each user completed
- /userstats → show user balance, referrals, withdrawals, and task count
- /withdrawstats → show pending/approved/rejected withdrawals
- /fundcheck → show whether bot has enough fund for pending withdrawals
